## Install

```sh
npm i @bobbykim/card-beta
```

## Usage

`import CardBeta from "@bobbykim/card-beta"`

In your code

```vue
<script setup lang="ts">
...
import CardBeta from '@bobbykim/card-beta'
...
</script>

<template>
  <card-beta></card-beta>
</template>
```
