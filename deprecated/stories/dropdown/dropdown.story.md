# Dropdown

## @bobbykim/mcl-dropdown

### Install

```sh
npm i @bobbykim/mcl-dropdown
```

### Usage

`import MclDropdown from "@bobbykim/mcl-dropdown"`;

In your code

```vue
<script setup lang="ts">
...
import MclDropdown from '@bobbykim/mcl-dropdown'
...
</script>

<template>
  <mcl-dropdown></mcl-dropdown>
</template>
```

### Dependencies

- @bobbykim/manguito-theme
