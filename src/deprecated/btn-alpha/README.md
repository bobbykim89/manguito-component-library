## Install

```sh
npm i @bobbykim/btn-alpha
```

## Usage

`import BtnAlpha from "@bobbykim/btn-alpha"`;

In your code

```vue
<script setup lang="ts">
...
import BtnAlpha from '@bobbykim/btn-alpha'
...
</script>

<template>
  <btn-alpha></btn-alpha>
</template>
```
