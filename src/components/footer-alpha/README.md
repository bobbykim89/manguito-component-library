## Install

```sh
npm i @bobbykim/footer-alpha
```

## Usage

`import FooterAlpha from "@bobbykim/footer-alpha"`;

In your code

```vue
<script setup lang="ts">
...
import FooterAlpha from '@bobbykim/footer-alpha'
...
</script>

<template>
  <footer-alpha></footer-alpha>
</template>
```
