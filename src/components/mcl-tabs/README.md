## Install

```sh
npm i @bobbykim/mcl-tabs
```

## Usage

`import { MclTabsA } from "@bobbykim/mcl-tabs"`

```vue
<script setup lang="ts">
...
import { MclTabsA } from "@bobbykim/mcl-tabs"
...
</script>

<template>
  <mcl-tabs-a></mcl-tabs-a>
</template>
```
