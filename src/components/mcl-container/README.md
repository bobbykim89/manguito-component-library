## Install

```sh
npm i @bobbykim/mcl-container
```

## Usage

`import { MclContainerA, MclContainerB } from "@bobbykim/mcl-container"`;

```vue
<script setup lang="ts">
...
import { MclContainerA, MclContainerB } from "@bobbykim/mcl-container"
...
</script>

<template>
  <mcl-container-a></mcl-container-a>
  <mcl-container-b></mcl-container-b>
</template>
```
