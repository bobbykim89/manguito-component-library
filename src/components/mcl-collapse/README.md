## Install

```sh
npm i @bobbykim/mcl-collapse
```

## Usage

`import { MclCollapseA, MclCollapseB } from "@bobbykim/mcl-collapse"`;

```vue
<script setup lang="ts">
...
import { MclCollapseA, MclCollapseB } from '@bobbykim/mcl-collapse'
...
</script>

<template>
  <mcl-collapse-a></mcl-collapse-a>
  <mcl-collapse-b></mcl-collapse-b>
</template>
```
