## Install

```sh
npm i @bobbykim/container-alpha
```

## Usage

`import ContainerAlpha from "@bobbykim/container-alpha"`;

In your code

```vue
<script setup lang="ts">
...
import ContainerAlpha from '@bobbykim/container-alpha'
...
</script>

<template>
  <container-alpha></container-alpha>
</template>
```
