## Install

```sh
npm i @bobbykim/btn-beta
```

## Usage

`import BtnBeta from "@bobbykim/btn-beta"`;

In your code

```vue
<script setup lang="ts">
...
import BtnBeta from '@bobbykim/btn-beta'
...
</script>

<template>
  <btn-beta></btn-beta>
</template>
```
