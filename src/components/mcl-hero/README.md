## Install

```sh
npm i @bobbykim/mcl-hero
```

## Usage

`import { MclHeroA, MclHeroB, MclHeroC } from "@bobbykim/mcl-hero"`;

```vue
<script setup lang="ts">
...
import { MclHeroA, MclHeroB, MclHeroC } from "@bobbykim/mcl-hero"
...
</script>

<template>
  <mcl-hero-a></mcl-hero-a>
  <mcl-hero-b></mcl-hero-b>
  <mcl-hero-c></mcl-hero-c>
</template>
```
