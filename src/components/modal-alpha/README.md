## Install

```sh
npm i @bobbykim/modal-alpha
```

## Usage

`import ModalAlpha from "@bobbykim/modal-alpha"`;

In your code

```vue
<script setup lang="ts">
...
import ModalAlpha from '@bobbykim/modal-alpha'
...
</script>

<template>
  <modal-alpha></modal-alpha>
</template>
```
