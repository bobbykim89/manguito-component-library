## Install

```sh
npm i @bobbykim/hero-beta
```

## Usage

`import HeroBeta from '@bobbykim/hero-beta'`

In your code

```vue
<script setup lang="ts">
...
import HeroBeta from '@bobbykim/hero-beta'
...
</script>

<template>
  <hero-beta></hero-beta>
</template>
```
