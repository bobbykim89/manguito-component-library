# @bobbykim/parallax-alpha

## Demo

## Install

```sh
npm i @bobbykim/parallax-alpha
```

## Usage

`import ParallaxAlpha from "@bobbykim/parallax-alpha"`;

In your code

```vue
<script setup lang="ts">
...
import ParallaxAlpha from '@bobbykim/parallax-alpha'
...
</script>

<template>
  <parallax-alpha></parallax-alpha>
</template>
```
