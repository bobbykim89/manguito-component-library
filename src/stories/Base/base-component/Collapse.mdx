# Collapse

Included in `@bobbykim/manguito-theme`

An animated collapsible panel with height and opacity transitions. The component owns its open/close state internally — control it through a template ref or through an `AccordionGroup` context.

```ts
import { AccordionGroup, Collapse } from '@bobbykim/manguito-theme'
```

---

## Usage

### Standalone

Hold a template ref and call `toggle()`, `open()`, or `close()` directly.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Collapse } from '@bobbykim/manguito-theme'

const collapseRef = ref<InstanceType<typeof Collapse>>()
</script>

<template>
  <button @click="collapseRef?.toggle()">Toggle</button>
  <Collapse ref="collapseRef" id="panel-1" class-name="p-4">
    Collapsible content here.
  </Collapse>
</template>
```

### Accordion with `AccordionGroup`

Wrap panels in `AccordionGroup` — only one panel can be open at a time. Each panel's `id` prop identifies it within the group.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { AccordionGroup, Collapse } from '@bobbykim/manguito-theme'

const panel1 = ref<InstanceType<typeof Collapse>>()
const panel2 = ref<InstanceType<typeof Collapse>>()
const panel3 = ref<InstanceType<typeof Collapse>>()
</script>

<template>
  <AccordionGroup>
    <div>
      <button @click="panel1?.toggle()">Panel 1</button>
      <Collapse ref="panel1" id="panel-1" class-name="p-4">Content 1</Collapse>
    </div>
    <div>
      <button @click="panel2?.toggle()">Panel 2</button>
      <Collapse ref="panel2" id="panel-2" class-name="p-4">Content 2</Collapse>
    </div>
    <div>
      <button @click="panel3?.toggle()">Panel 3</button>
      <Collapse ref="panel3" id="panel-3" class-name="p-4">Content 3</Collapse>
    </div>
  </AccordionGroup>
</template>
```

### Accordion with `v-for`

Store refs in an array when panels are generated from data.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { AccordionGroup, Collapse } from '@bobbykim/manguito-theme'

const items = [
  { id: 'panel-1', title: 'Section 1', content: 'Content for section 1.' },
  { id: 'panel-2', title: 'Section 2', content: 'Content for section 2.' },
  { id: 'panel-3', title: 'Section 3', content: 'Content for section 3.' },
]

const panelRefs = ref<InstanceType<typeof Collapse>[]>([])
</script>

<template>
  <AccordionGroup>
    <div v-for="(item, index) in items" :key="item.id">
      <button @click="panelRefs[index]?.toggle()">{{ item.title }}</button>
      <Collapse
        :ref="(el) => { if (el) panelRefs[index] = el as InstanceType<typeof Collapse> }"
        :id="item.id"
        class-name="p-4"
      >
        {{ item.content }}
      </Collapse>
    </div>
  </AccordionGroup>
</template>
```

## API

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `id` | `string` | auto-generated | Panel identifier used by `AccordionGroup` to track which panel is open. Also set as the element's HTML `id` attribute for ARIA wiring. |
| `visible` | `boolean` | `false` | Initial open state. Reactive — changing it after mount opens or closes the panel. |
| `class-name` | `string \| string[]` | `''` | CSS class(es) applied to the inner content wrapper. |

### Slots

| Slot | Description |
|---|---|
| `default` | Collapsible content. |

### Exposed Methods

| Method | Signature | Description |
|---|---|---|
| `toggle` | `() => void` | Toggle open/closed. |
| `open` | `() => void` | Open the panel. |
| `close` | `() => void` | Close the panel. |

### Events

| Event | Payload | Description |
|---|---|---|
| `open` | `(visible: boolean)` | Emitted when the panel opens. |
| `close` | `(visible: boolean)` | Emitted when the panel closes. |
