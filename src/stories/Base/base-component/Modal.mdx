# Modal

Included in `@bobbykim/manguito-theme`

A dialog overlay with focus trap, scroll lock, backdrop, and animated open/close transition. ARIA attributes (`role="dialog"`, `aria-modal`, `aria-labelledby`) are applied automatically.

```ts
import { Modal } from '@bobbykim/manguito-theme'
```

---

## Usage

### Open via component ref

The recommended way to trigger a modal is to hold a template ref and call `open()`.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Modal } from '@bobbykim/manguito-theme'

const modalRef = ref<InstanceType<typeof Modal>>()
</script>

<template>
  <button @click="modalRef?.open()">Open Modal</button>

  <Modal ref="modalRef" id="my-modal" title="My Modal">
    <template #body>
      <p>Modal content goes here.</p>
    </template>
  </Modal>
</template>
```

### Open via `:visible` prop

Bind a reactive ref to `:visible` for state-driven control.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Modal } from '@bobbykim/manguito-theme'

const isOpen = ref(false)
</script>

<template>
  <button @click="isOpen = true">Open Modal</button>

  <Modal :visible="isOpen" @close="isOpen = false" id="my-modal">
    <template #body="{ close }">
      <p>Modal content goes here.</p>
      <button @click="close">Close</button>
    </template>
  </Modal>
</template>
```

### Custom header, body, and footer

Use named slots to fully customise the modal layout. All slot props expose `close()` and `status`.

```vue
<template>
  <Modal ref="modalRef" id="my-modal" color="light-1" class-name="rounded-md">
    <template #header="{ close }">
      <div class="flex justify-between p-4">
        <h3>Title</h3>
        <button @click="close" aria-label="Close">✕</button>
      </div>
    </template>
    <template #body="{ close, status }">
      <p>Status: {{ status }}</p>
      <button @click="close">Close from body</button>
    </template>
    <template #footer="{ close }">
      <div class="flex justify-end p-4">
        <button @click="close">Done</button>
      </div>
    </template>
  </Modal>
</template>
```

### Default header

Pass `title` to use the built-in header instead of `#header` slot.

```vue
<Modal ref="modalRef" id="my-modal" title="Built-in Header" color="primary">
  <template #body>...</template>
</Modal>
```

## API

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `id` | `string` | — | HTML `id` on the dialog root. Used for ARIA `aria-labelledby` wiring. |
| `title` | `string` | — | Text for the built-in default header. Omit when using `#header` slot. |
| `titleColor` | `ColorPalette` | `'dark-3'` | Color of the default header title text. |
| `visible` | `boolean` | `false` | Initial open state. Reactive — changing it opens or closes the modal. |
| `color` | `ColorPalette` | `'light-1'` | Background color of the modal panel. |
| `class-name` | `string \| string[]` | `''` | Additional CSS class(es) on the modal panel. |
| `placement` | `'top' \| 'center' \| 'bottom'` | `'center'` | Vertical alignment of the modal on screen. |
| `modal-width` | `'small' \| 'medium' \| 'large'` | `'small'` | Width preset for the modal panel. |
| `no-backdrop` | `boolean` | `false` | When `true`, renders without the semi-transparent backdrop. |
| `no-header` | `boolean` | `false` | When `true`, suppresses the header area entirely. |
| `backdrop-color` | `ColorPalette` | `'dark-3'` | Color of the backdrop overlay. |

### Slots

All three slots receive `{ close: () => void, status: boolean }` as slot props.

| Slot | Slot props | Description |
|---|---|---|
| `#header` | `{ close, status }` | Custom header. Omit to use the default header (requires `title` prop) or `no-header`. |
| `#body` | `{ close, status }` | Main content area. |
| `#footer` | `{ close, status }` | Footer area below the body. |

### Exposed Methods

| Method | Signature | Description |
|---|---|---|
| `toggle` | `() => void` | Toggle open/closed. |
| `open` | `() => void` | Open the modal. |
| `close` | `() => void` | Close the modal. |

### Events

| Event | Payload | Description |
|---|---|---|
| `open` | `(visible: boolean)` | Emitted when the modal opens. |
| `close` | `(visible: boolean)` | Emitted when the modal closes. |
