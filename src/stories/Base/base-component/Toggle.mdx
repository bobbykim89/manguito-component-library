# Toggle Directive — `v-toggle`

> **Deprecated.** `v-toggle` communicates state through DOM attribute mutation, bypassing Vue's reactivity system. It remains functional on `Modal` and `Sidebar` during the transition period but will be removed in a future release.

---

## Migration

Use component refs and `@click` handlers instead:

```vue
<!-- Before (deprecated) -->
<script setup lang="ts">
import { vToggle } from '@bobbykim/manguito-theme/directives'
</script>

<template>
  <button v-toggle:my-modal>Open</button>
  <Modal id="my-modal">...</Modal>
</template>
```

```vue
<!-- After — preferred: component ref -->
<script setup lang="ts">
import { ref } from 'vue'
import { Modal } from '@bobbykim/manguito-theme'

const modalRef = ref<InstanceType<typeof Modal>>()
</script>

<template>
  <button @click="modalRef?.open()">Open</button>
  <Modal ref="modalRef">...</Modal>
</template>
```

```vue
<!-- After — preferred: :visible prop binding -->
<script setup lang="ts">
import { ref } from 'vue'
import { Modal } from '@bobbykim/manguito-theme'

const isOpen = ref(false)
</script>

<template>
  <button @click="isOpen = true">Open</button>
  <Modal :visible="isOpen" @close="isOpen = false">...</Modal>
</template>
```
