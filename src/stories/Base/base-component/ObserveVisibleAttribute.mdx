# observeVisibleAttr

> **Internal composable.** This composable is an implementation detail of the `v-toggle` and `v-collapse` directives. It is not intended for direct use in application code. Both directives are deprecated and will be removed in a future release, after which this composable will be removed as well.

---

## Background

`observeVisibleAttr` uses a `MutationObserver` to watch for changes to a `visible` DOM attribute and convert them into reactive Vue state. It was introduced as a bridge between the directive-based state communication pattern (`vToggle`, `vCollapse`) and Vue's reactivity system.

The directive pattern has been superseded by composable-based state management (`useCollapseState`, `AccordionGroup`) and the `defineExpose` API on `Modal` and `Sidebar`.

---

## API

```ts
import { observeVisibleAttr } from '@bobbykim/manguito-theme/composable'

observeVisibleAttr(
  el: Ref<HTMLElement | undefined>,
  callback: (visible: boolean) => void,
): { stop: () => void } | undefined
```

| Parameter | Type | Description |
|---|---|---|
| `el` | `Ref<HTMLElement \| undefined>` | Template ref of the element to observe. |
| `callback` | `(visible: boolean) => void` | Called when the `visible` attribute changes. Receives `true` if `visible="true"`, `false` otherwise. |

Returns `{ stop: () => void }` to disconnect the observer manually, or `undefined` if the element ref is not yet populated. The observer disconnects automatically on component unmount.

---

## Preferred alternatives

For toggle/visibility state in new code, use Vue's reactivity directly:

```vue
<!-- Simple toggle -->
<script setup lang="ts">
import { ref } from 'vue'
const isOpen = ref(false)
</script>
<template>
  <button @click="isOpen = !isOpen">Toggle</button>
  <div v-if="isOpen">Content</div>
</template>
```

For Modal and Sidebar, use component refs:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { Modal } from '@bobbykim/manguito-theme'

const modalRef = ref<InstanceType<typeof Modal>>()
</script>
<template>
  <button @click="modalRef?.open()">Open</button>
  <Modal ref="modalRef">...</Modal>
</template>
```

For collapse accordion groups, use `AccordionGroup` + `Collapse`:

```ts
import { AccordionGroup, Collapse } from '@bobbykim/manguito-theme'
```

See the [Collapse docs](?path=/docs/base-base-component-collapse-component--collapse) and [Modal docs](?path=/docs/base-base-component-modal-component--modal) for complete examples.
