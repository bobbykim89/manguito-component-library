# List class

## Default list

```html
<ul class="mcl-list">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```

```html
<ul class="mcl-list mcl-list-primary">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>

<ul class="mcl-list mcl-list-secondary">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>

<ul class="mcl-list mcl-list-info">
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```
