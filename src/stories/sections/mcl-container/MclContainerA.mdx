# MCL Container A

Included in `@bobbykim/mcl-container`

## Installation

```sh
npm i @bobbykim/mcl-container
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclContainerA } from '@bobbykim/mcl-container'
</script>

<template>
  <mcl-container-a></mcl-container-a>
</template>
```
