## Slot

- `#content-right`: slot on the right side of the component
- `#mobile-bottom(props: { headerClose: () => void })`: slot at the bottom of collapse menu on mobile screen

## Events

- `toggle-drawer` - (event: Event, open: boolean) - event fired on drawer button click
- `title-click` - (event: Event, url: string, target: CtaTarget) - event fired on title block (title, logo) click
- `menu-click` - (event: Event, item: MenuItemType) - event fired on menu item click
- `collapse-click` - (event: Event, title: string, visible: boolean) - event fired on collapsable menu label click
