# MCL Collapse A

Included in `@bobbykim/mcl-collapse`

## Installation

```sh
npm i @bobbykim/mcl-collapse
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclCollapseA } from '@bobbykim/mcl-collapse'
</script>

<template>
  <mcl-collapse-a></mcl-collapse-a>
</template>
```

## Events

- `collapse-open` - (visible: boolean, title: string) - Emitted when collapse component opens
- `collapse-close` - (visible: boolean, title: string) - Emitted when collapse component closes

## Slots

- `#tab`: slot in the right side of accordion head, does not affected by default accordion click behavior (toggle)
- `#content`: slot in the content area
