# MCL Collapse B

Included in `@bobbykim/mcl-collapse`

## Installation

```zsh
npm i @bobbykim/mcl-collapse
```

## Usage

In your code

```js
<script setup lang="ts">
import {MclCollapseB} from '@bobbykim/mcl-collapse'

</script>

<template>
  <mcl-collapse-b></mcl-collapse-b>
</template>
```
