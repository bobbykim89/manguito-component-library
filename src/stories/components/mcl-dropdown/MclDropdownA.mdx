## Events

-`button-click`: (event: Event) - Event fired on dropdown button click.

-`item-click`: (event: Event, item: DropdownItem) - Event fired on dropdown item click.

## Keyboard accessibility

-`escape`: closes dropdown

-`arrow up`: focus dropdown item below (no effect if the last dropdown item is selected)

-`arrow down`: focus dropdown item above (no effect if the first dropdown item is focused)

## types

```js
export interface DropdownItem {
  title: string
  [key: string]: string
}
export interface ItemClickEvent {
  event: Event
  item: DropdownItem
}
```
