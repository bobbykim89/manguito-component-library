# MCL Input File

Included in `@bobbykim/mcl-forms`

## Installation

```sh
npm i @bobbykim/mcl-forms
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclInputFile } from '@bobbykim/mcl-forms'
</script>

<template>
  <mcl-input-file v-model="inputData"></mcl-input-file>
</template>
```
