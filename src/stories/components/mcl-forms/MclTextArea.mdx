# MCL Text Area

Included in `@bobbykim/mcl-forms`

## Installation

```sh
npm i @bobbykim/mcl-forms
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclTextArea } from '@bobbykim/mcl-forms'
</script>

<template>
  <mcl-text-area v-model="inputData"></mcl-text-area>
</template>
```
