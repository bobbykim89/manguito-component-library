# MCL Input Switch

Included in `@bobbykim/mcl-forms`

## Installation

```sh
npm i @bobbykim/mcl-forms
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclInputSwitch } from '@bobbykim/mcl-forms'
</script>

<template>
  <mcl-input-switch v-model="inputData"></mcl-input-switch>
</template>
```
