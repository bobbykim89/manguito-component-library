# MCL Select

Included in `@bobbykim/mcl-forms`

## Installation

```sh
npm i @bobbykim/mcl-forms
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclSelect } from '@bobbykim/mcl-forms'
</script>

<template>
  <mcl-select v-model="inputData"></mcl-select>
</template>
```
