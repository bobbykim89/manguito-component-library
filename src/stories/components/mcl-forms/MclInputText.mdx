# MCL Input Text

Included in `@bobbykim/mcl-forms`

## Installation

```sh
npm i @bobbykim/mcl-forms
```

## Usage

In your code

```js
<script setup lang="ts">
import { MclInputText } from '@bobbykim/mcl-forms'
</script>

<template>
  <mcl-input-text v-model="inputData"></mcl-input-text>
</template>
```
